# Glossário SQL
## -- 
Comentário
## INTEGER 
Número Inteiro (Int)
## TEXT 
Texto (String)
## PRIMARY KEY 
Chave que não se repete, usada para identificar uma entidade
## AUTOINCREMENT 
O SQLite gera o dado automaticamente
## FOREIGN KEY
 Usa uma chave primária de outro atributo para atribuir um ao outro
